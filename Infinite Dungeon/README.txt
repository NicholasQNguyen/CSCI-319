Infinite Dungeon

Controls:
Keyboard
    WASD for movement
    Arrow Keys to shoot
Controller
(Tested with an XBox Series X Controller. But should in theory work for 
an XBox One or an XBox 360 controller. Or it should work with any
controller with a dpad and 4 buttons)
    DPad for movement
    Face buttons to shoot

How to start:
Run main.py
